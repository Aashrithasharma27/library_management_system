# Library Management System (SQL)

A relational database project built in MySQL to simulate a library's book lending system.

## Tables
- **Novels** — book catalog (title, author, genre, published year, copies available)
- **Members** — library members (name, email, join date)
- **Loan** — tracks who borrowed which book and when

## Skills Practiced
- Table design with primary/foreign keys
- Joins across 3 tables
- Aggregate functions (COUNT, AVG)
- GROUP BY and HAVING
- Subqueries
- Debugging real SQL errors (column/alias mismatches)

- List all books currently on loan
- Find the most borrowed book
